# 🌍 GeoInsight

GeoInsight is an intelligent geospatial analysis platform that leverages data, AI, and visualization to evaluate land suitability, environmental trends, and urban development insights.

---

## 🚀 Features

* 📍 Location-based analysis using geospatial data
* 🌱 Environmental insights (NDVI, land health, canopy analysis)
* 📊 Data visualization for better decision-making
* ⚠️ Alerts & monitoring for environmental risks
* 📱 Cross-platform support (Web + Mobile)

---

## 🧠 Tech Stack

### Frontend / Mobile

* React Native (Expo)
* TypeScript

### Backend

* Node.js / Express (or your backend stack)

### AI / Data

* Geospatial analysis (NDVI, land classification)
* Environmental data processing

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/WannaLearnCodingForFun/GeoInsight.git
cd GeoInsight
```

---

### 2. Install dependencies

#### Backend

```bash
cd backend
npm install
```

#### Mobile App

```bash
cd mobile
npm install
```

---

### 3. Run the project

#### Backend

```bash
npm start
```

#### Mobile (Expo)

```bash
npx expo start
```
